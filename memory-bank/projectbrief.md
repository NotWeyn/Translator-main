# Project Brief: Arch Linux Screen Translator

## Core Requirements
- **OS**: Arch Linux (specifically Hyprland/Wayland/KdePlasma(etc.) environment).
- **Functionality**:
  - Screen text detection (OCR).
  - Context integrity check (clustering nearby text).
  - Text correction (spell check + AI/LLM based).
  - Translation (Multi-language support).
  - Overlay: Display translated text over the original text on screen.
- **UI**: Modern, user-friendly interface for configuration.
- **Customization**:
  - Selectable OCR backends (EasyOCR, PaddleOCR).
  - Selectable Translation backends (OpenAI API, Local models, etc.).
  - Configurable correction settings.

## Goals
Create a seamless, high-performance "screen translation" tool that feels native to the Linux gaming/desktop experience, solving the problem of fragmented text in games and applications.
